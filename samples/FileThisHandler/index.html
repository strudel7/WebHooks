ASP.NET WebHooks FileThis Handler

This sample illustrates how to wire up a FileThis WebHooks receiver. A sample WebHook URI is:

https://<host>/api/webhooks/incoming/filethis/{id}

Set the 'MS_WebHookReceiverSecret_FileThis' application setting to the application secret, optionally using IDs to differentiate between multiple WebHooks, for example 'secret0, id1=secret1, id2=secret2'.

Please see FileThis Developer Docs for more information.
